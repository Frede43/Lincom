# 👨‍🏫 CAS D'UTILISATION - MENTOR/INSTRUCTEUR

## Parcours Utilisateur Complet

### 🎯 **1. ONBOARDING & PROFIL**

#### UC-M001: Devenir mentor
**Acteur:** Expert souhaitant partager ses connaissances
**Objectif:** Rejoindre la plateforme comme mentor
**Pages nécessaires:**
- Formulaire candidature mentor
- Page upload CV/portfolio
- Page définition expertise
- Page validation par admin
- Page onboarding mentor

**Flux:**
```
Candidature → CV/Portfolio → Expertise → Validation → Onboarding → Dashboard Mentor
```

#### UC-M002: Configurer son profil mentor
**Acteur:** Mentor accepté
**Objectif:** Optimiser sa visibilité
**Pages nécessaires:**
- Page profil mentor détaillé
- Page tarification et disponibilités
- Page spécialités et compétences
- Page portfolio de réussites

### 📚 **2. CRÉATION DE CONTENU**

#### UC-M003: Créer un cours
**Acteur:** Mentor expert
**Objectif:** Partager ses connaissances via un cours
**Pages nécessaires:**
- Interface création de cours (CMS)
- Éditeur de contenu multimédia
- Page structuration modules/leçons
- Page création quiz/évaluations
- Page preview et publication

**Flux:**
```
Dashboard → Nouveau cours → Structure → Contenu → Quiz → Preview → Publication
```

#### UC-M004: Gérer ses cours
**Acteur:** Mentor avec cours publiés
**Objectif:** Maintenir et améliorer ses formations
**Pages nécessaires:**
- Dashboard des cours créés
- Analytics de performance cours
- Page gestion des étudiants inscrits
- Interface de mise à jour contenu
- Page feedback et reviews

### 🤝 **3. MENTORAT INDIVIDUEL**

#### UC-M005: Gérer les demandes de mentorat
**Acteur:** Mentor disponible
**Objectif:** Sélectionner et accompagner des mentees
**Pages nécessaires:**
- Page demandes de mentorat reçues
- Profils détaillés des candidats
- Interface d'acceptation/refus
- Page planning des sessions
- Contrats de mentorat

**Flux:**
```
Demandes reçues → Profil candidat → Acceptation → Planning → Session
```

#### UC-M006: Conduire des sessions de mentorat
**Acteur:** Mentor avec mentees assignés
**Objectif:** Fournir un accompagnement de qualité
**Pages nécessaires:**
- Interface de visioconférence
- Page notes de session partagées
- Outils de collaboration (whiteboard)
- Page suivi progression mentee
- Interface d'évaluation

### 🏆 **4. ÉVALUATION & CERTIFICATION**

#### UC-M007: Évaluer les étudiants
**Acteur:** Mentor/Instructeur
**Objectif:** Valider les acquis des apprenants
**Pages nécessaires:**
- Interface de correction quiz/devoirs
- Page attribution de notes
- Générateur de certificats
- Page feedback détaillé
- Dashboard de performance classe

#### UC-M008: Certifier les compétences
**Acteur:** Mentor expert équipements
**Objectif:** Valider l'utilisation sécurisée des machines
**Pages nécessaires:**
- Page formations équipements
- Interface d'évaluation pratique
- Générateur de certifications
- Page suivi validité certifications

### 📊 **5. ANALYTICS & PERFORMANCE**

#### UC-M009: Suivre ses performances
**Acteur:** Mentor actif
**Objectif:** Optimiser son impact
**Pages nécessaires:**
- Dashboard analytics mentor
- Page revenus et paiements
- Métriques d'engagement étudiants
- Page reviews et ratings reçus
- Rapport d'impact mensuel

### 🎪 **6. ÉVÉNEMENTS & WORKSHOPS**

#### UC-M010: Organiser des événements
**Acteur:** Mentor reconnu
**Objectif:** Animer la communauté
**Pages nécessaires:**
- Interface création d'événement
- Page gestion des inscriptions
- Outils de diffusion en direct
- Page feedback post-événement
- Analytics de participation
