Authentication API Documentation

Security Note

All authentication endpoints use HTTPS. Tokens are valid for 24 hours. Refresh tokens are valid for 7 days.

Authentication

POST /api/auth/login/

Authenticate user and receive access token

Request Body:
{
    "email": "user@example.com",
    "password": "secure_password"
}
Response:
{
    "access_token": "eyJ0eXAiOiJKV1QiLCJhbGciOiJIUzI1NiJ9...",
    "refresh_token": "eyJ0eXAiOiJKV1QiLCJhbGciOiJIUzI1NiJ9...",
    "token_type": "Bearer",
    "expires_in": 86400
}
POST /api/auth/refresh/

Get new access token using refresh token

Request Body:
{
    "refresh_token": "eyJ0eXAiOiJKV1QiLCJhbGciOiJIUzI1NiJ9..."
}
POST /api/auth/logout/

Invalidate current access token

User Management

POST /api/auth/register/

Register a new user account

Request Body:
{
    "email": "user@example.com",
    "password": "secure_password",
    "confirm_password": "secure_password",
    "first_name": "John",
    "last_name": "Doe",
    "user_type": "startup",  // or "organization"
    "phone_number": "+25779123456"
}
POST /api/auth/verify-email/

Verify user email address

Request Body:
{
    "token": "verification_token"
}
POST /api/auth/reset-password-request/

Request password reset email

Request Body:
{
    "email": "user@example.com"
}
POST /api/auth/reset-password/

Reset user password

Request Body:
{
    "token": "reset_token",
    "new_password": "new_secure_password",
    "confirm_password": "new_secure_password"
}

Profile Management

GET /api/auth/profile/

Get current user profile

Response:
{
    "id": 1,
    "email": "user@example.com",
    "first_name": "John",
    "last_name": "Doe",
    "user_type": "startup",
    "phone_number": "+25779123456",
    "profile_picture": "https://example.com/profile.jpg",
    "created_at": "2024-01-01T00:00:00Z",
    "last_login": "2024-01-15T12:30:00Z"
}
PUT /api/auth/profile/

Update user profile

Request Body:
{
    "first_name": "John",
    "last_name": "Doe",
    "phone_number": "+25779123456",
    "profile_picture": "base64_encoded_image"
}
POST /api/auth/change-password/

Change user password

Request Body:
{
    "current_password": "current_password",
    "new_password": "new_secure_password",
    "confirm_password": "new_secure_password"
}

Security Settings

POST /api/auth/2fa/enable/

Enable two-factor authentication

Response:
{
    "secret_key": "JBSWY3DPEHPK3PXP",
    "qr_code": "base64_encoded_qr_code"
}
POST /api/auth/2fa/verify/

Verify two-factor authentication code

Request Body:
{
    "code": "123456"
}
POST /api/auth/2fa/disable/

Disable two-factor authentication
