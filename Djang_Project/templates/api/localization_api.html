Localization & Internationalization API Documentation

Supported Languages

Available languages: Français English Kirundi

Language Management

POST /api/localization/languages/

Add new language support

Request Body:
{
    "code": "rn",
    "name": "Kirundi",
    "native_name": "Ikirundi",
    "direction": "ltr",
    "region": "bi",
    "fallback": "fr",
    "enabled": true,
    "metadata": {
        "translator": "Jean Ndayishimiye",
        "last_updated": "2024-01-19T14:00:00Z"
    }
}
Response:
{
    "id": "lang_123",
    "code": "rn",
    "name": "Kirundi",
    "native_name": "Ikirundi",
    "status": "active",
    "translation_progress": 0,
    "created_at": "2024-01-19T14:00:00Z"
}
GET /api/localization/languages/

List supported languages

Query Parameters:
Parameter Type Description
status string Filter by status
region string Filter by region

Translation Management

POST /api/localization/translations/

Add new translations

Request Body:
{
    "language": "rn",
    "namespace": "common",
    "translations": {
        "welcome": "Murakaza neza",
        "login": "Kwinjira",
        "signup": "Kwiyandikisha",
        "dashboard": {
            "title": "Ikibaho",
            "summary": "Incamake"
        }
    },
    "metadata": {
        "version": "1.0",
        "reviewed": false
    }
}
GET /api/localization/translations/export/

Export translations

Query Parameters:
Parameter Type Description
language string Language code
format string Export format (json, yaml, po)

Content Localization

POST /api/localization/content/

Localize content

Request Body:
{
    "content_type": "article",
    "content_id": "article-123",
    "language": "rn",
    "fields": {
        "title": "Intangamarara",
        "description": "Ibisobanuro birambuye",
        "content": "# Intangamarara\nMurakaza neza...",
        "metadata": {
            "keywords": ["ubuhinga", "ikoranabuhanga"]
        }
    },
    "status": "draft"
}
GET /api/localization/content/{content_type}/{content_id}/

Get localized content

Regional Settings

PATCH /api/localization/settings/

Update regional settings

Request Body:
{
    "date_format": "DD/MM/YYYY",
    "time_format": "HH:mm",
    "timezone": "Africa/Bujumbura",
    "currency": {
        "code": "BIF",
        "symbol": "FBu",
        "position": "after"
    },
    "number_format": {
        "decimal_separator": ",",
        "thousand_separator": ".",
        "decimals": 2
    }
}

Translation Memory

POST /api/localization/memory/

Add to translation memory

Request Body:
{
    "source_language": "fr",
    "target_language": "rn",
    "source_text": "Bienvenue sur notre plateforme",
    "target_text": "Murakaza neza kuri platform yacu",
    "context": "welcome_message",
    "metadata": {
        "domain": "tech",
        "quality": 100
    }
}
GET /api/localization/memory/search/

Search translation memory

Query Parameters:
Parameter Type Description
q string Search query
source_lang string Source language
target_lang string Target language
