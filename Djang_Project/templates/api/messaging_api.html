Messaging & Chat API Documentation

Message Types

Available types: Message Direct Message Groupe Diffusion

Direct Messaging

POST /api/messages/direct/

Send a direct message

Request Body:
{
    "recipient_id": "user-123",
    "content": "Bonjour! Comment allez-vous?",
    "attachments": [
        {
            "file_id": "file-789",
            "type": "image"
        }
    ],
    "metadata": {
        "reply_to": "message-456",
        "mentions": ["user-789"]
    }
}
Response:
{
    "id": "message-123",
    "sender": {
        "id": "user-456",
        "name": "Alice Smith",
        "avatar_url": "https://example.com/avatars/alice.jpg"
    },
    "recipient": {
        "id": "user-123",
        "name": "Bob Johnson"
    },
    "content": "Bonjour! Comment allez-vous?",
    "sent_at": "2024-01-19T14:00:00Z",
    "status": "sent"
}
GET /api/messages/conversations/

List user conversations

Query Parameters:
Parameter Type Description
status string Filter by status (active, archived)
unread boolean Filter unread conversations

Group Chat

POST /api/messages/groups/

Create a group chat

Request Body:
{
    "name": "Équipe Technique",
    "description": "Groupe pour l'équipe technique",
    "members": ["user-123", "user-456", "user-789"],
    "avatar": "file-123",
    "settings": {
        "allow_member_invites": true,
        "message_retention_days": 30
    }
}
POST /api/messages/groups/{group_id}/messages/

Send a message to a group

Request Body:
{
    "content": "Réunion d'équipe demain à 10h",
    "attachments": [
        {
            "file_id": "file-789",
            "type": "document"
        }
    ],
    "announcement": false
}

Broadcast Messages

POST /api/messages/broadcast/

Send a broadcast message

Request Body:
{
    "title": "Maintenance Planifiée",
    "content": "Le système sera en maintenance le 20 janvier",
    "target": {
        "roles": ["admin", "manager"],
        "departments": ["tech", "operations"],
        "locations": ["Bujumbura"]
    },
    "schedule": {
        "send_at": "2024-01-19T18:00:00Z",
        "expire_at": "2024-01-21T00:00:00Z"
    },
    "options": {
        "require_acknowledgment": true,
        "priority": "high"
    }
}
GET /api/messages/broadcasts/

List broadcast messages

Message Actions

POST /api/messages/{message_id}/reactions/

Add reaction to message

Request Body:
{
    "emoji": "👍",
    "type": "like"
}
POST /api/messages/{message_id}/forward/

Forward a message

Request Body:
{
    "recipients": ["user-123", "group-456"],
    "note": "Information importante"
}

Message Search

GET /api/messages/search/

Search messages

Query Parameters:
Parameter Type Description
q string Search query
type string Message type
date_range string Date range
Response:
{
    "results": [
        {
            "id": "message-123",
            "type": "direct",
            "content": "Bonjour! Comment allez-vous?",
            "sender": {
                "id": "user-456",
                "name": "Alice Smith"
            },
            "sent_at": "2024-01-19T14:00:00Z",
            "context": {
                "conversation_id": "conv-789",
                "highlight": "Bonjour! Comment allez-vous?"
            }
        }
    ],
    "total": 15,
    "page": 1,
    "per_page": 10
}
