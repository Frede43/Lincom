API Documentation v1.0

Authentication

All API endpoints require authentication via JWT token. Include the token in the Authorization header:

Authorization: Bearer <your_token>

Organizations

GET /api/organizations/

Retrieve a list of all organizations

Query Parameters:
Parameter Type Description
page integer Page number for pagination
size integer Number of items per page 
{
    "count": 100,
    "next": "http://api.example.org/organizations/?page=2",
    "previous": null,
    "results": [
        {
            "id": 1,
            "name": "Organization Name",
            "description": "Organization Description",
            "website": "https://example.org",
            "created_at": "2024-01-01T00:00:00Z"
        }
    ]
}
GET /api/organizations/{id}/

Retrieve details of a specific organization

{
    "id": 1,
    "name": "Organization Name",
    "description": "Organization Description",
    "website": "https://example.org",
    "created_at": "2024-01-01T00:00:00Z",
    "metrics": {
        "total_project_calls": 10,
        "active_project_calls": 5,
        "total_competitions": 8,
        "active_competitions": 3
    }
}

Project Calls

GET /api/project-calls/

Retrieve a list of all project calls

Query Parameters:
Parameter Type Description
status string Filter by status (draft, published, closed, cancelled)
organization integer Filter by organization ID
POST /api/project-calls/

Create a new project call

Request Body:
{
    "title": "Project Call Title",
    "description": "Project Call Description",
    "start_date": "2024-01-01",
    "end_date": "2024-12-31",
    "budget": 1000000,
    "requirements": "Project requirements..."
}

Competitions

GET /api/competitions/

Retrieve a list of all competitions

Query Parameters:
Parameter Type Description
type string Filter by type (pitch, hackathon, challenge)
status string Filter by status (upcoming, ongoing, completed, cancelled)
POST /api/competitions/{id}/register/

Register a startup for a competition

Request Body:
{
    "startup_id": 1,
    "team_members": ["John Doe", "Jane Smith"],
    "additional_info": "Additional registration information..."
}

Analytics

GET /api/analytics/organization/{id}/metrics/

Retrieve analytics metrics for an organization

Query Parameters:
Parameter Type Description
date_range string Time period for metrics (7d, 30d, 90d, 1y) 
{
    "participation_rate": 75.5,

    "average_score": 4.2,

    "active_projects": 25,
    "trends": {
        "participation": 5.2,
        "acceptance": -2.1,
        "score": 0.3
    }
}
