Settings & Configuration API Documentation

Configuration Management

Manage system-wide settings, organization preferences, and feature flags. All changes are audited and versioned.

General Settings

GET /api/settings/

Get all system settings

Response:
{
    "general": {
        "site_name": "Community Lab Burundi",
        "contact_email": "contact@example.com",
        "timezone": "Africa/Bujumbura",
        "default_language": "fr",
        "maintenance_mode": false
    },
    "branding": {
        "logo_url": "https://example.com/logo.png",
        "favicon_url": "https://example.com/favicon.ico",
        "primary_color": "#3498db",
        "secondary_color": "#2ecc71"
    },
    "features": {
        "enable_registration": true,
        "enable_oauth": true,
        "enable_2fa": true
    }
}
PUT /api/settings/

Update system settings (Admin only)

Request Body:
{
    "general": {
        "site_name": "Community Lab Burundi",
        "maintenance_mode": true
    },
    "features": {
        "enable_registration": false
    }
}

Email Settings

PUT /api/settings/email/

Configure email settings (Admin only)

Request Body:
{
    "smtp": {
        "host": "smtp.example.com",
        "port": 587,
        "username": "noreply@example.com",
        "password": "smtp-password",
        "use_tls": true
    },
    "sender": {
        "name": "Community Lab Burundi",
        "email": "noreply@example.com"
    },
    "templates": {
        "welcome_email": {
            "subject": "Bienvenue à {{site_name}}",
            "enabled": true
        },
        "reset_password": {
            "subject": "Réinitialisation du mot de passe",
            "enabled": true
        }
    }
}
POST /api/settings/email/test/

Send test email

Request Body:
{
    "to": "test@example.com",
    "template": "welcome_email"
}

Feature Flags

GET /api/settings/features/

List all feature flags

Response:
{
    "features": [
        {
            "key": "new_dashboard",
            "name": "New Dashboard Interface",
            "description": "Enable the new dashboard UI",
            "enabled": true,
            "type": "boolean",
            "rollout_percentage": 100
        },
        {
            "key": "ai_suggestions",
            "name": "AI Suggestions",
            "description": "Enable AI-powered suggestions",
            "enabled": true,
            "type": "gradual",
            "rollout_percentage": 50
        }
    ]
}
PUT /api/settings/features/{feature_key}/

Update feature flag settings

Request Body:
{
    "enabled": true,
    "rollout_percentage": 75,
    "target_users": ["premium", "beta"],
    "conditions": {
        "country": ["Burundi"],
        "device_type": ["desktop", "mobile"]
    }
}

List available languages

Audit Log

GET /api/settings/audit/

Get settings change history

Query Parameters:
Parameter Type Description
section string Filter by settings section
user string Filter by user who made changes
Response:
{
    "changes": [
        {
            "id": "change-123",
            "section": "general",
            "key": "site_name",
            "old_value": "Tech Hub",
            "new_value": "Community Lab Burundi",
            "changed_by": "admin@example.com",
            "timestamp": "2024-01-19T14:00:00Z",
            "reason": "Rebranding update"
        }
    ]
}
