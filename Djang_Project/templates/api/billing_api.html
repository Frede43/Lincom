Subscription Management

POST /api/billing/subscriptions/

Create a new subscription

Request Body:
{
    "plan_id": "plan_pro_monthly",
    "organization_id": "org-123",
    "payment_method": {
        "type": "card",
        "token": "pm_card_visa"
    },
    "billing_address": {
        "street": "123 Rue du Commerce",
        "city": "Bujumbura",
        "country": "Burundi",
        "postal_code": "12345"
    },
    "tax_id": "TAX123456789",
    "coupon": "WELCOME2024"
}
Response:
{
    "id": "sub_123",
    "plan": {
        "id": "plan_pro_monthly",
        "name": "Pro Monthly",
        "price": 4900,  // in cents
        "currency": "USD"
    },
    "status": "active",
    "current_period_end": "2024-02-19T14:00:00Z",
    "cancel_at_period_end": false
}
PUT /api/billing/subscriptions/{subscription_id}/

Update subscription

Request Body:
{
    "plan_id": "plan_pro_yearly",
    "quantity": 10,
    "payment_method_id": "pm_card_mastercard",
    "cancel_at_period_end": false
}
POST /api/billing/subscriptions/{subscription_id}/cancel/

Cancel subscription

Request Body:
{
    "reason": "budget_constraints",
    "feedback": "Service excellent mais trop cher pour nous",
    "cancel_immediately": false
}
