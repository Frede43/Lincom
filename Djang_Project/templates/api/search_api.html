Search & Filtering API Documentation

Search Capabilities

The search API supports full-text search, fuzzy matching, and advanced filtering options. Results are ranked by relevance score.

Global Search

GET /api/search/

Search across all content types

Query Parameters:
Parameter Type Description
q string Search query
type string Filter by content type (startup, project, event)
sort string Sort order (relevance, date, name)
Response:
{
    "total_results": 125,
    "execution_time": 0.234,
    "results": {
        "startups": [
            {
                "id": 1,
                "type": "startup",
                "name": "TechStart Burundi",
                "description": "Innovation technologique",
                "relevance_score": 0.95,
                "highlight": {
                    "name": "TechStart Burundi",
                    "description": "Innovation technologique"
                }
            }
        ],
        "projects": [...],
        "events": [...]
    }
}
GET /api/search/autocomplete/

Get search suggestions

Query Parameters:
Parameter Type Description
q string Partial search query
limit integer Maximum number of suggestions

Advanced Search

POST /api/search/advanced/

Perform advanced search with complex filters

Request Body:
{
    "query": "tech startup",
    "filters": {
        "type": ["startup", "project"],
        "sector": ["Technology", "Education"],
        "location": "Bujumbura",
        "date_range": {
            "start": "2024-01-01",
            "end": "2024-12-31"
        },
        "custom_fields": {
            "team_size": {"min": 5, "max": 20},
            "funding_stage": ["seed", "series_a"]
        }
    },
    "sort": [
        {"field": "relevance", "order": "desc"},
        {"field": "date", "order": "desc"}
    ],
    "page": 1,
    "per_page": 20
}

POST /api/search/save/

Save a search query

Request Body:
{
    "name": "Tech Startups in Bujumbura",
    "query": "tech startup",
    "filters": {
        "location": "Bujumbura",
        "sector": "Technology"
    },
    "notify": true
}

Saved Searches

GET /api/search/saved/

List saved searches

Response:
{
    "saved_searches": [
        {
            "id": 1,
            "name": "Tech Startups in Bujumbura",
            "query": "tech startup",
            "filters": {
                "location": "Bujumbura",
                "sector": "Technology"
            },
            "notify": true,
            "created_at": "2024-01-19T14:00:00Z",
            "last_executed": "2024-01-19T15:00:00Z"
        }
    ]
}
DELETE /api/search/saved/{id}/
