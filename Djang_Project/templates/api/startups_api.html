Startups API Documentation

Startup Management

GET /api/startups/

Retrieve a list of all startups

Query Parameters:
Parameter Type Description
sector string Filter by business sector
stage string Filter by startup stage (idea, early, growth, mature)
location string Filter by location
search string Search in name and description

POST /api/startups/

Create a new startup

Request Body:
Field Type Description
name Required string Name of the startup
description Required string Detailed description of the startup
sector Required string Business sector
stage Required string Current stage of the startup
team_size Optional integer Number of team members 
{
    "name": "TechStart Burundi",
